**Project Title: Aryan Gyming Website**
**Purpose: To provide an online presence for Aryan Gyming, detailing services, membership plans, and a contact platform for visitors.**

Features:
Home Page:

A welcoming interface showcasing the Aryan Gym logo and navigation menu.
Menu links to key sections: About Us, Services, Contact Us.
Contact Page:

Includes a form for user inquiries, collecting details like name, email, phone number, and questions.
Integrated Google Maps iframe for easy location identification.
Displays essential contact details such as address, email, and phone.
Services Page:

Highlights available membership plans:
Monthly Plan: ₹1500 per month.
Three-Monthly Plan: ₹4000 for three months (₹500 discount).
Yearly Plan: ₹10,000 per year (₹8,000 discount).
Features a structured layout using stylized sections for each plan.
Design Elements:

Consistent theme with a black background, bold white text, and accent colors for headings and highlights.
Custom fonts and icons enhance the professional appearance.
Responsive design for compatibility with various devices.
Additional Elements:

Professional styling with CSS.
Linked external resources such as Font Awesome icons and Google Fonts for enhanced aesthetics.
Favicon and logo to reinforce branding.
Technology Used:
HTML: Structure of the website.
CSS: Custom styles and layouts.
External Libraries: Font Awesome for icons and Google Fonts for typography.
Google Maps: Embed functionality for location services.
User Benefits:
Simplified navigation to learn about gym services and membership plans.
Direct access to contact details and an inquiry form.
Transparent pricing details for memberships.
Enhancements:
Potential improvements could include integrating dynamic features like user authentication, membership management, or testimonials from existing members.
